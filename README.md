# RunCalculator

This is a modern React app scaffolded with Vite and TypeScript.

## Getting Started

### Install dependencies
```
npm install
```

### Start the development server
```
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).

## Project Structure
- `src/` — Main source code (React components, styles, etc.)
- `.github/` — Project instructions and automation

## Next Steps
