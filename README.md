# RunCalculator

This is a modern React app scaffolded with Vite and TypeScript.

## CI Status

![CI - Tests and Coverage](https://github.com/dannelinder/RunCalculator/actions/workflows/tests.yml/badge.svg)

## Getting Started

### Install dependencies
```
npm install
```

### Start the development server
```
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).

## Project Structure
- `src/` — Main source code (React components, styles, etc.)
- `.github/` — Project instructions and automation

## Testing

- Run tests: `npm run test`
- Watch tests: `npm run test:watch`
- Coverage: `npm run test:coverage`

## Deployment

- Vercel: `vercel` then `vercel --prod`
- GitHub Pages: add `gh-pages` and set Vite `base` accordingly
